# MedCore

MedCore is a comprehensive Patient Health Portal application built with Flutter. It allows users to manage their medical records, view lab results, track vitals, and manage prescriptions.

## Features

- **Home Dashboard**: Aggregated view of Vitals, Reminders, and Active Prescriptions.
- **Medical History**: Chronological timeline of diagnosis and pharmacy events.
- **Lab Results**: Detailed visualization of lab reports with range sliders and status flags.
- **Upload Records**: Flow to upload and digitize new medical documents.
- **Reminders**: Medication and appointment scheduling.

## Getting Started

1.  **Prerequisites**: Flutter SDK `3.x`
2.  **Installation**:
